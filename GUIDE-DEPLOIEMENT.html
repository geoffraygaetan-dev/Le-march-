🧺 Guide de déploiement

Mettre Le Marché en ligne gratuitement — environ 15 minutes

Tu vas créer un projet Firebase (base de données temps réel, gratuit) et le déployer sur Vercel (hébergement gratuit). Résultat : une URL publique que tu partages avec ta copine, accessible sur tous les téléphones.

ÉTAPE 1

📦 Télécharger le code du projet

Tu as reçu un fichier marche-app.zip. Dézippe-le sur ton ordinateur, tu obtiendras un dossier avec cette structure :

marche-app/
  src/
    App.jsx   ← toute l'application
    firebase.js  ← à remplir à l'étape 2
    main.jsx
  index.html
  package.json
  vite.config.js
Tu n'as PAS besoin d'installer quoi que ce soit sur ton ordinateur pour le déploiement — Vercel s'occupe de tout !
ÉTAPE 2

🔥 Créer ta base de données Firebase

2a) Va sur console.firebase.google.com et connecte-toi avec ton compte Google.

2b) Créer un projet

Clique "Ajouter un projet" → donne-lui un nom (ex: le-marche-2024) → désactive Google Analytics si proposé → clique "Créer le projet"

2c) Activer la base de données temps réel

Dans le menu gauche : Créer → Realtime Database → clique "Créer une base de données" → choisis la région europe-west1 (Belgique) → sélectionne "Démarrer en mode test" → Activer

2d) Récupérer la configuration

Menu gauche : ⚙️ Paramètres du projet → Général → descends jusqu'à "Vos applications" → clique "</> Web" → donne un nom (ex: marche-web) → Enregistrer l'application. Tu verras alors un bloc de code comme ceci :

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "le-marche-2024.firebaseapp.com",
  databaseURL: "https://le-marche-2024-default-rtdb.europe-west1.firebasedatabase.app",
  projectId: "le-marche-2024",
  storageBucket: "le-marche-2024.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc..."
};
2e) Coller la config dans firebase.js

Ouvre le fichier src/firebase.js dans un éditeur de texte (Notepad, TextEdit…). Remplace chaque valeur COLLE_TON_…_ICI par les vraies valeurs copiées depuis Firebase. Sauvegarde le fichier.

La databaseURL est indispensable. Elle ressemble à https://mon-projet-default-rtdb.europe-west1.firebasedatabase.app
ÉTAPE 3

🐙 Mettre le code sur GitHub

Vercel a besoin que ton code soit sur GitHub pour le déployer.

3a) Créer un compte GitHub

Va sur github.com → Sign up → crée un compte gratuit (si tu n'en as pas)

3b) Créer un dépôt

Clique "New repository" → nom : le-marche → laisse tout par défaut → "Create repository"

3c) Uploader les fichiers

Sur la page du dépôt vide, clique "uploading an existing file" → glisse-dépose tout le contenu du dossier marche-app (tous les fichiers et le dossier src) → descends en bas → clique "Commit changes"

Tu dois voir dans GitHub les fichiers : index.html, package.json, vite.config.js, et le dossier src/
ÉTAPE 4

🚀 Déployer sur Vercel

4a) Créer un compte Vercel

Va sur vercel.com → "Sign Up" → connecte-toi avec ton compte GitHub (c'est plus simple)

4b) Importer le projet

Clique "Add New → Project" → tu verras ton dépôt le-marche → clique "Import"

4c) Configurer et déployer

Vercel détecte automatiquement que c'est un projet Vite. Tu n'as rien à changer. Clique simplement "Deploy". Attends 1-2 minutes…

Vercel te donnera une URL du type le-marche.vercel.app — c'est l'adresse de ton application !
ÉTAPE 5

🔒 Sécuriser la base de données (important !)

Le mode test Firebase expire après 30 jours. Pour que ça marche définitivement, modifie les règles de sécurité.

5a) Ouvrir les règles Firebase

Dans la console Firebase → Realtime Database → Règles

Remplace les règles existantes par celles-ci :

{
  "rules": {
    "marche": {
      ".read": true,
      ".write": true
    }
  }
}

Clique "Publier".

Ces règles permettent à toi et ta copine de lire/écrire sans compte. Seule l'URL de ton app peut accéder aux données marche.
✅ TERMINÉ

📱 Partager avec ta copine

Envoie simplement l'URL Vercel à ta copine (ex: le-marche.vercel.app). Elle ouvre le lien sur son téléphone, et c'est synchronisé instantanément !

Ajouter en raccourci sur l'écran d'accueil

Sur iPhone : Safari → bouton Partager → "Sur l'écran d'accueil"
Sur Android : Chrome → menu ⋮ → "Ajouter à l'écran d'accueil"
L'app se comportera comme une vraie application mobile !

🎉 Ton application est en ligne !
